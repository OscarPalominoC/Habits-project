# Habits RPG — Sube de Nivel en la Vida Real ⚔️

**Habits RPG** es una aplicación de seguimiento de hábitos gamificada inspirada en las mecánicas de los RPG modernos y el sistema de "Solo Leveling". Transforma tu rutina diaria en una épica búsqueda de poder, donde cada tarea completada te otorga experiencia (XP), te permite subir de nivel y desbloquear títulos legendarios.

## 🌟 Características Principales

- **Sistema de Misiones**: Crea hábitos diarios, semanales o personalizados con metas específicas.
- **Progresión RPG**: Gana XP por cada actividad completada. Sube de nivel para transformarte de un "Noob" a un "Monarca del Sistema".
- **Identidad Única**: Personaliza tu perfil con un selector de Avatares dinámico (Mago, Guerrero, Ninja, etc.).
- **Rachas de Fuego (Streaks)**: Mantén tu disciplina para aumentar tu racha de días consecutivos y ver arder tu contador de llamas.
- **Historial Detallado**: Registra tu progreso con checklists interactivas y visualiza tu evolución histórica.

## 🏗️ Estructura del Proyecto

El proyecto está dividido en dos grandes módulos:

- **/backend**: API de alto rendimiento construida con FastAPI y MongoDB. Gestiona la lógica de juego, autenticación y persistencia de datos.
- **/frontend**: Interfaz de usuario inmersiva construida con React, Vite y Tailwind CSS, utilizando Framer Motion para animaciones cinemáticas.

## 🚀 Inicio Rápido

### Requisitos Previos
- Node.js (v18+)
- Python (3.10+)
- MongoDB (Local o Atlas)

### Configuración del Backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # o .venv\Scripts\activate en Windows
pip install -r requirements.txt
python main.py
```

### Configuración del Frontend
```bash
cd frontend
npm install
npm run dev
```
